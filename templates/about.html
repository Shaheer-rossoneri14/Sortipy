About Sortipy

Sort your Spotify playlists by any of a wide range of musical attributes such as tempo, loudness, valence, energy, danceability and more.


With Sortipy you can easily order the songs in any of your playlists on a wide range of parameters. Just follow these steps:

  1. Copy your playlist URL.
  2. Paste the URL and click on Submit button.
  3. Scroll Down to find the buttons to sort the playlist.
  4. Sort the playlist by clicking on the button.


FAQ

Here are some answers to questions about Sortipy

  1. How was this built?
    2. This was created using the Spotify API using Python on the backend.
  2. Is the source code available?
    3. Yes, you can find it on GitHub
  3. Does Sortipy overwrite my playlist when I save?
    4. Only if you chose the 'Overwrite playlist option' otherwise it creates a copy of the playlist. The new playlist is called 'Old playlist name sorted by increasing X', where X is what you sorted the playlist on.
